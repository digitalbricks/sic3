# Versioncheck Addon by "Slugger"
The idea and the majority of code for this addon was contributed by the user ["Slugger" in the WBCE CMS forum](https://forum.wbce.org/viewtopic.php?pid=48034#p48034) in June 2026. It was the "ignition spark" for adding the addon system to SIC 3 in version 3.4.0.

## What it does
This addon allows for comparing the versions of CMS on sites registered inside SIC3 with the latest available version of the used CMS. So it gives you a glimpse on sites who needs an update. CMS not directly supported by the addon can be added via [extensions](extensions/README.md) 

The language of the module is german only.
